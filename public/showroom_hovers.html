1. L'Inversion Brutaliste

Style Galerie d'Art & Haute Couture

Inspiré du minimalisme haut de gamme. Aucun dégradé ni ombre portée complexe. L'effet de hover se produit via un contraste net et immédiat de type "négatif/positif". La carte blanche devient noire, le texte devient blanc, et l'indicateur de direction glisse subtilement vers la droite.

Comportement & Intégration

  • Inversion instantanée ou ultra-rapide (180ms)
  • Aucun arrondi de boîte (angles bruts orthogonaux)
  • Translation horizontale de la flèche de navigation
  • Adapté pour conserver le look "Galerie Éditoriale" du site

Propriétés CSS Clés

.card {
  transition: all 0.18s ease-in-out;
}
.card:hover {
  background-color: #111;
  color: #fff;
  border-color: #111;
}
